
# DR@ESCRIBA - MVP Visual Moderno

Site com estilo preto + dourado, fonte tech (Orbitron) e botão direto para WhatsApp.

## Como usar

1. Suba os arquivos no seu repositório GitHub
2. Ative o GitHub Pages (Settings > Pages > Source: main / root)
3. Acesse seu site pelo link gerado

## Estrutura
- `index.html` - Home com botão WhatsApp
- `contato.html` - Contato
- `style.css` - Estilo
